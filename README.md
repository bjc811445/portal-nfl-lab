# portal-nfl-lab

## 项目简介

`portal-nfl-lab` 是一个用于归档和发布多个独立 HTML 页面的仓库。本项目不针对任何特定域名或网站，仅作为静态页面内容的存储与展示空间。每个页面均为独立的 HTML 文件，可单独访问或引用。

## 目录说明

```
portal-nfl-lab/
├── index.html          # 仓库主页（可选）
├── pages/              # 存放所有独立 HTML 页面
│   ├── page-01.html
│   ├── page-02.html
│   └── ...
├── assets/             # 静态资源文件（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

## 页面归档说明

- 所有 HTML 页面存放于 `pages/` 目录下，文件名应简短且具有辨识度。
- 每个页面均为自包含的完整 HTML 文档，可直接在浏览器中打开。
- 若页面依赖外部资源（如样式表、脚本或图片），请将资源文件放在 `assets/` 对应子目录中，并在 HTML 中使用相对路径引用。
- 仓库不承担对页面内容进行审核或修改的责任，仅作为存档与分发渠道。

## 维护说明

- 欢迎提交 Pull Request 添加新的独立 HTML 页面，或修正现有页面中的错误。
- 提交新页面时，请确保文件格式正确、资源引用有效，并避免包含任何指向外部网站的链接。
- 不建议在此仓库中存放大量重复或无关页面，以保持结构清晰。
- 本仓库由维护者不定期更新，不承诺固定的发布周期。

## 注意事项

- 本仓库仅用于存档 HTML 页面，不提供任何形式的服务或功能。
- 请勿将本仓库用于违反法律法规或侵犯他人权益的内容。
- 所有页面的内容与用途由贡献者自行负责。
